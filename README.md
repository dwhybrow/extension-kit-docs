# Extension Kit Docs (Markdown Mirror)
A curated markdown more of [Unit4 Extension Kit public documentation](https://docs-external.u4pp.com/extensions-kit/), maintained for use with AI tooling and agent development.

## Purpose
The official docs are HTML-based and not easily consumable by AI assistants or automated agents. This repo provides clean markdown versions of selected pages for use as context in Claude Projects, AI Foundry knowledge bases, or any agent that can reference raw GitHub content.

## Source
All content is derived from publicly accessible documentation. This repo does not contain any proprietry or client-specific information.

## Maintenance
Pages are manually reviewed and updated periodically. Check `release-notes.md` for the latest version captured
