# Message Hub Event Action

## Description

A _Message Hub event message action_ allows a flow to send a Message Hub event message. If the event message does not exist it will be created.

## Usage

Create a flow with the desired trigger type and add Message Hub Event Action.

The mandatory input parameters for this action are the event name and version (tenant comes by default). You can also add JSON body and properties.

![Image 4: Message Hub Event Action](https://docs-external.u4pp.com/extensions-kit/images/built-in-actions/mh-event-action-new.png)

The output shows whether the action has been completed successfully or not.
