# Unit4Id Resolver

## Description

In the Unit4 Ecosystem services often relies on the so called _Unit4Id_, that is an identifier _Unit4 Identity Services_ uses to uniquely identify an user for a given tenant.

This action allows us to look up that identifier, based on the a-mail of a given user, returning additionally the name of the given user.

![Image 3: Unit4 Id Resolver](https://docs-external.u4pp.com/extensions-kit/images/built-in-actions/unit4Id-resolver.png)

> Note: This action can only resolve the _Unit4 Id_ for users that are known by ExtensionsKit.
