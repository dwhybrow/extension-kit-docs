# Stop Execution

## Description

The _Stop Execution_ action is used to stop the execution of a flow, even if there are additional steps defined after this action in the flow.

## Usage

It is usually used in combination with a _Condition_, so the flow will be only stopped in case of the condition is met, otherwise the Flow will continue its execution. The _Stop Execution_ action also controls the status of the execution of the flow, via the property _Stop with_.
