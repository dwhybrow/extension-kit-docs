# Pause

## Description

This action allows users to temporarily **pause a Flow execution** for a specified amount of time.

![Image 3: Pause Action example configuration](https://docs-external.u4pp.com/images/built-in-actions/pause-configuration.png)

## Usage

After selecting the action you can set the duration of the pause:

*   Select an amount for Hours (between 00 and 23)
*   Select an amount for Minutes (between 00 and 59)

> **NOTE**: The minimum amount of time required for the action is `1 minute` and the maxium is `23 Hours and 59 minutes`.
